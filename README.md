# gmod_mtm
In-game map thumbnail maker.

## Commands
- `mtm_start` - Starts capturing
- `mtm_stop` - Stops capturing

## ConVars
- `mtm_size` - Map thumbnail size (in pixels)
- `mtm_format` - Map thumbnail image format (png, jp, jpeg)
